# Blogger

A simple and clean Django blogging web application.

This project demonstrates core Django concepts:
- Template inheritance
- Dynamic URL routing
- ORM-based data handling
- Form handling with `POST`
- Responsive UI using Bootstrap 5

## Features

- View latest published posts on the home page
- Create a new post from a form page
- Read full post content on a dedicated detail page
- Responsive card layout for posts
- Shared navbar and footer via a base template

## Tech Stack

- Python
- Django 6
- SQLite (default Django database)
- Bootstrap 5 (CDN)
